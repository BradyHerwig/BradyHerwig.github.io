# Project screenshots

Store showcase images here. Recommended layout:

```
screenshots/
  my-project-slug/
    01-overview.png
    02-detail.png
```

## Tips

- Prefer WebP or compressed PNG/JPEG
- Aim for ~1280–1600px wide for crisp 16:10 frames
- Use descriptive filenames
- After adding files, tell the agent the project name, slug, and which images go where so they can wire them into `projects.html`
